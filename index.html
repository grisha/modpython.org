It's Alive!

Aug 23, 2013

Mod_python is an Apache module that embeds the Python interpreter within the server. With mod_python you can write web-based applications in Python that will run many times faster than traditional CGI and will have access to advanced features such as ability to retain database connections and other data between hits and access to Apache internals. A more detailed description of what mod_python can do is available in this O'Reilly article.

Current State of Mod_Python

Mod_python has not been under active development in the recent years. I am trying to address this situation by bringing it up to speed with the latest versions of Python and Apache. At this point there is no specific timeline on when the next version might be released.

The board of the Apache Software Foundation has placed mod_python in the attic, which essentially means that the ASF is no longer maintaining this project. Thus any and all development work on mod_python will be outside of the ASF and its infrastructure. The exact implications of this aren't very clear to me yet, I am going to try to keep them minimal.
