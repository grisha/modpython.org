Well, I think I used the Debian source package, which builds mod_python
like this:

        ./configure --with-apxs=/usr/bin/apxs2 --prefix=/usr \
                --mandir=\$${prefix}/share/man --infodir=\$${prefix}/share/info \
                --with-python-version=$(PYVER)
