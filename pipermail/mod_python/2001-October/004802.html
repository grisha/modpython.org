It looks like my httpd.conf was wrong. I was following description from
http://www.modpython.org/live/mod_python-2.7.6/doc-html/app-wininst.html but
I didn't read comments in section AddModule of my httpd.conf: [WHENEVER YOU
CHANGE THE LOADMODULE SECTION ABOVE, UPDATE THIS TOO!]

When I added 'AddModule mod_python.c' Apache can be restarted and mptest.py
works.

I think would be nice to mention this in Windows Installation document, just
in case.

Thanks for all comments and hints,
