Here is the first one (and most complex):

def test():                                                              
    start = time()                                              
    tab = {}                                            
    fp = open("/home/kbryd/public_html/python2/data", "r")                       
    n = 0                                                                
    lines = fp.readlines()                                       
    for line in lines:                                   
	fields = split(line, '=')                
        tab[n] = [fields[0], fields[1]]  
        n+=1                     
    fp.close()   
    end = time()                                                                
    return end-start                                                    

(the file is 400KB long)

It was repeated 100 times to calculate average.
