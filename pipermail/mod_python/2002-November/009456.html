Hi Jaime

You can write your own error message to the client, and return
apache.DONE to stop Apache from any further processing.

E.g.

def handler(req):
  try:
    <Do some processing>
  except:
    <Write error message, including headers>
    return apache.DONE

  return apache.OK


Cheers!

Conrad
