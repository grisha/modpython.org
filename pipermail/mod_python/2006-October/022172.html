You may be able to get away with this:


  grabber = StringIO()
  oldwrite = req.write
  req.write = grabber.write
  try:
      doSomePSPStuff(req)
  finally:
      req.write = oldwrite


This would redirect any output in the doSomePSPStuff function into the 
grabber object.
