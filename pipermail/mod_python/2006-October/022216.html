I use a ConnectionHandler in order to implement the persistent Z39.50
binary protocol within mod_python.

For example:

Listen *:2100
<VirtualHost *:2100>
   PythonPath "['/home/cheshire/cheshire3/code']+sys.path"
   PythonConnectionHandler zApacheHandler
   PythonDebug On
</VirtualHost>
