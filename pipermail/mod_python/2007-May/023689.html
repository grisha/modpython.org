StringField,str and other basic type was allowed to be pickled.

Looking at 3.3.1 util.py :

     329             # make a Field
     330             if disp_options.has_key("filename"):

     335                 field.name = name
     336             field.file = file

Why always assign the "file" member ? The docs says it works on demand
