<FONT COLOR="#000000">BTW, you are using &quot;index&quot; for name of function suggesting you are</FONT>
<FONT COLOR="#000000">using publisher. If you are using publisher, are you using SetHandler</FONT>
<FONT COLOR="#000000">or AddHandler?</FONT>

<FONT COLOR="#000000">If you use SetHandler so that all requests to the directory go through</FONT>
<FONT COLOR="#000000">publisher, it will mean that your stylesheet request will also go through</FONT>
<FONT COLOR="#000000">publisher. It will not be able to satisfy the request and will return not</FONT>
<FONT COLOR="#000000">found. Browsers tend to ignore not found errors on stylesheets and</FONT>
<FONT COLOR="#000000">do not show an error.</FONT>

<FONT COLOR="#000000">Thus, if you must use SetHandler, include in your configuration:</FONT>

<FONT COLOR="#000000">  &lt;Files *.css&gt;</FONT>
<FONT COLOR="#000000">  SetHandler default-handler</FONT>
<FONT COLOR="#000000">  &lt;/Files&gt;</FONT>

<FONT COLOR="#000000">Graham</FONT>
