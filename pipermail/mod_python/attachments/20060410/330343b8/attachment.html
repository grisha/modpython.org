<P><FONT SIZE=2 FACE="Arial">I am probably remiss for not having shared this before now.&nbsp; Here are two output filters for mod_python that I have developed.</FONT></P>

<P><FONT SIZE=2 FACE="Arial">The first filter, xsltfilter.py, transforms XML into XHTML (or whatever else you may desire) via an XSLT template.&nbsp; The second filter, xincludefilter.py, processes xinclude elements imbedded in XHTML.&nbsp; Both are meant as a simple means of providing &quot;quasi-dynamic&quot; web content.&nbsp; Benchmarks on my hardware show throughputs of 100 to 500+ pages per second using these filters.</FONT></P>

<P><FONT SIZE=2 FACE="Arial">Dependencies:</FONT>

<BR><FONT SIZE=2 FACE="Arial">lxml:&nbsp; an ElemenTree-like binding for the libxml2/libxslt libraries (see </FONT><A HREF="http://www.codespeak.net/lxml"><U><FONT COLOR="#0000FF" SIZE=2 FACE="Arial">http://www.codespeak.net/lxml</FONT></U></A><FONT SIZE=2 FACE="Arial">)</FONT>
</P>

<P><FONT SIZE=2 FACE="Arial">In the &quot;main&quot; Apache config:</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">AddHandler mod_python .py</FONT>

<BR><FONT SIZE=2 FACE="Courier New">PythonPath &quot; ...(as needed)... &quot;</FONT>

<BR><FONT SIZE=2 FACE="Courier New">PythonOutputFilter xsltfilter XSLTFILTER</FONT>

<BR><FONT SIZE=2 FACE="Courier New">PythonOutputFilter xincludefilter XINCLUDEFILTER</FONT>
</P>

<P><FONT SIZE=2 FACE="Arial">In the appropriate Apache VHOST section:</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">AddType text/html .xml .xhtml</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">&lt;Directory &quot; &#8230;(as needed)&#8230; &quot;&gt;</FONT>

<BR>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <FONT SIZE=2 FACE="Courier New">AddOutputFilter XSLTFILTER .xml</FONT>

<BR>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <FONT SIZE=2 FACE="Courier New">AddOutputFilter XINCLUDEFILTER .xhtml</FONT>

<BR>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <FONT SIZE=2 FACE="Courier New">&#8230;.</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&lt;/Directory&gt;</FONT>
</P>

<P><FONT SIZE=2 FACE="Arial">File xsltfilter.py:&nbsp; (Note - Could probably stand some refactoring)</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">from mod_python import apache</FONT>

<BR><FONT SIZE=2 FACE="Courier New">import lxml.etree</FONT>

<BR><FONT SIZE=2 FACE="Courier New">from cStringIO import StringIO</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">xsltfile = open('/path/to/my/template.xslt', 'rU')</FONT>

<BR><FONT SIZE=2 FACE="Courier New">xsltstring = xsltfile.read()</FONT>

<BR><FONT SIZE=2 FACE="Courier New">xsltfile.close()</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">styledoc = lxml.etree.parse(StringIO(xsltstring))</FONT>

<BR><FONT SIZE=2 FACE="Courier New">transformer = lxml.etree.XSLT(styledoc)</FONT>
</P>

<P><FONT SIZE=2 FACE="Courier New">def outputfilter (filter):</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; xmlstring = filter.read()</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; doc = lxml.etree.parse(StringIO(xmlstring))</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; doc.xinclude()</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; result = str(transformer(doc))</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; filter.write(result)</FONT>

<BR><FONT SIZE=2 FACE="Courier New">&nbsp;&nbsp;&nbsp; filter.close()</FONT>
</P>
