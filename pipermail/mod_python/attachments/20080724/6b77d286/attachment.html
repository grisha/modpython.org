<p class=MsoPlainText>I think I found an answer to my own question.<o:p></o:p></p>

<p class=MsoPlainText>On AMD x86_64, when I am linking a static library into a
shared library, position-dependent code in a shared library is not supported.<o:p></o:p></p>

<p class=MsoPlainText>So I recompiled python with --enable-shared option, so
that there is no static library libpython2.4.a at all, and it worked.<o:p></o:p></p>

<p class=MsoPlainText><o:p>&nbsp;</o:p></p>

<p class=MsoPlainText>I wonder if --enable-shared option causes any performance
