<DIV><FONT face=Arial size=2>In mod_python, form data is returned as a mapping 
type (dictionary) instead of a sequence type (tuple or list.)</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>A very handy way to get the form data is to import 
the util module from mod python:</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>&nbsp;&nbsp;&nbsp; <FONT face="Courier New">from 
mod_python import util</FONT></FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>and then within your handler instantiate an object 
of the FieldStorage Class:</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; def 
myhandler(req):</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; 
...(stuff)...</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; 
formdata = util.FieldStorage(req)</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; 
...(stuff)...</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>You can then get any of the form's values by 
name.&nbsp; Say your form is...</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &lt;input 
name="age"...</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &lt;input 
name="gender"...</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &lt;input 
name="shoe_size"...</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>...you can then get all your form data from the 
formdata object:</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; if formdata['gender'] == 
'male':</FONT></DIV>
<DIV><FONT face="Courier New" size=2>&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; 
oddness_coefficient = formdata['age']&nbsp;+ formdata['shoe_size']</FONT></DIV>
<DIV><FONT face="Courier New" size=2></FONT>&nbsp;</DIV>
<DIV><FONT face=Arial size=2>In my opinion,&nbsp;accessing your form data by 
name instead of by sequence&nbsp;offsets makes your code a LOT easier to 
read.</FONT></DIV>
<DIV><FONT face=Arial size=2></FONT>&nbsp;</DIV>
