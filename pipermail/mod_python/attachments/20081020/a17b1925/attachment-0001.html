<p>Job Responsibilities</p>
<p>The Sr. Software Developer will be responsible for the development of the WordStream suite of products. Development is done primarily using Python on the server, and with Adobe Flex or HTML/AJAX as a Rich Internet Application (RIA) frontend. Typical day-to-day activities include implementing new features in our products, which involve:</p>

<p>&nbsp;&nbsp;&nbsp; * Development of processes to extract relevant data from input sources<br>&nbsp;&nbsp;&nbsp; * Design of relational data storage models<br>&nbsp;&nbsp;&nbsp; * Implementation of data processing algorithms<br>&nbsp;&nbsp;&nbsp; * Development of AMF remoting methods, used for client/server connectivity<br>
&nbsp;&nbsp;&nbsp; * Development of graphical user interfaces</p>
<p>Qualifications</p>
<p>The candidate must have top-notch software development skills. Prior experience in Python or Adobe Flex is not a strict requirement, experience in related technologies will be accepted. Please note, only candidates with a bachelor&#39;s degree in computer science or engineering, or, a very substantial amount of related technical coursework and work experience in a related field will be considered. Prior experience working on web analytics applications, search engines, content management systems or other Web publishing systems is a plus.</p>

<p>Required Skills: Structured Query Language (SQL), Object-Oriented Programming</p>
<p>Bonus Skills: Python, Adobe Flex, SQLite, Open-Source software development, Git, Object-Relational Mapping (ORM), Postgres SQL, AJAX, HTML, Search Engine Optimization (SEO) or Search Engine Marketing (SEM)</p>
<p>Type of Position: Full-time or Contract-to-hire</p>

<p>Position Title: Sr. Software Developer</p>
<p>Compensation and Benefits: Competitive salary based on experience.&nbsp; Employees will have an opportunity to participate in the company&#39;s stock option plan.</p>
