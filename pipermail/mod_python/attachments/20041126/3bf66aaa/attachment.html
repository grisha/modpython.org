You can skip using util.fieldstorage altogether if you use req.form which has already done all the fieldstorage stuff for you and is a dict of the form values with the field names as keys<BR>
<BR>

    <FONT COLOR="#737373" SIZE="2"><I>Greetings!</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">In mod_python, form data is returned as a mapping type (dictionary) instead of a sequence type (tuple or list.)</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">A very handy way to get the form data is to import the util module from mod python:</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">&nbsp;&nbsp;&nbsp; from mod_python import util</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">and then within your handler instantiate an object of the FieldStorage Class:</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">&nbsp;&nbsp;&nbsp; def myhandler(req):<BR>
    &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ...(stuff)...<BR>
    &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; formdata = util.FieldStorage(req)<BR>
    &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; ...(stuff)...</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">You can then get any of the form's values by name.&nbsp; Say your form is...</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">&nbsp;&nbsp;&nbsp; &lt;input name=&quot;age&quot;...<BR>
    &nbsp;&nbsp;&nbsp; &lt;input name=&quot;gender&quot;...<BR>
    &nbsp;&nbsp;&nbsp; &lt;input name=&quot;shoe_size&quot;...</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">...you can then get all your form data from the formdata object:</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">&nbsp;&nbsp;&nbsp; if formdata['gender'] == 'male':<BR>
    &nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; oddness_coefficient = formdata['age']&nbsp;+ formdata['shoe_size']</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">In my opinion,&nbsp;accessing your form data by name instead of by sequence&nbsp;offsets makes your code a LOT easier to read.</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">I hope this is of some help.</FONT><BR>
    <FONT COLOR="#737373">&nbsp;</FONT><BR>
    <FONT COLOR="#737373" SIZE="2">Best Regards,<BR>
    Lee E. Brown</FONT><BR>
