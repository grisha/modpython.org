I used to get the same error before I set things up so that mod_python
PACKAGE is found.<br>
<br>
Make sure you mod_python Python PACKAGE (not the DLL files) and make sure
that PYTHONPATH environmental variable points to the directory where the
package is located. The mod_python package is a set of python files in a
mod_python directory and a bunch of subdirectories.<br>
<br>
For example, if you put the package in C:\python_packages (so that you
now have c:\python_packages\mod_python\...) make sure PYTHONPATH has
C:\python_packages in it. Then restart Apache.<br>
<br>
What you want to do is define PYTHONPATH in your environment BEFORE
starting the Apache (so that Apache process inherits it). You can then
also do in httpd.conf:<br>
<br>
PassEnv PYTHONPATH<br>
<br>
though I don't think it makes a difference.<br>
<br>
Hope this helps.<br>
