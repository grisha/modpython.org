The only way to do this safely and in a portable way would be to store
the request object in a thread-local variable (see
<a href="http://aspn.activestate.com/ASPN/Cookbook/Python/Recipe/302088">http://aspn.activestate.com/ASPN/Cookbook/Python/Recipe/302088</a> ).
Thread locals are only present in Python 2.4, though (but a pure Python
implementation is provided and nothing prevents it from running on
previous versions of Python).<br>
