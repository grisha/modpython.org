OK so Wayne, you may get what you want right now by setting the ApplicationPath option :<br>
<br>
<font style="font-family: courier new,monospace;" size="2">PythonOption ApplicationPath /path/to/my/app/as/an/url</font><br>
<br>
For example if app1 is accessible through <a href="http://localhost/app1">http://localhost/app1</a> and
app2 accessible through <a href="http://localhost/app2">http://localhost/app2</a>, then you would define <span style="font-family: courier new,monospace;">PythonOption ApplicationPath /app1</span> in a Directory section or in a .htaccess file for your first app, and 
<span style="font-family: courier new,monospace;">PythonOption ApplicationPath /app2</span> for your second app.<br>
