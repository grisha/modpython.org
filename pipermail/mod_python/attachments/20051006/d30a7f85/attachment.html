Hi Ken,<br><br>Unfortunately for a lot of mod_python beginners, it is
not clear why the mptest handler CANNOT be used directly by the
publisher handler.<br><br>First of all, try to have the publisher running as described here :<br><a href="http://www.modpython.org/live/current/doc-html/tut-pub.html">http://www.modpython.org/live/current/doc-html/tut-pub.html
</a><br><br>Then, you'll understand that even if the publisher is correctly configured, if you call <span style="font-family: courier new,monospace;">/mod_python/mptest.py</span>, then the publisher handler will find the mptest module but NOT an 
<span style="font-family: courier new,monospace;">index(req)</span> function inside... You'll have to call <span style="font-family: courier new,monospace;">/mod_python/mptest.py/handler</span> to specify the function to call (or rename 
<span style="font-family: courier new,monospace;">handler </span>to <span style="font-family: courier new,monospace;">index</span>) and begin to solve your 404 problem.<br><br>Last, but not least, handlers are supposed to write on the request object and return a status value (hence the 
<span style="font-family: courier new,monospace;">return apache.OK</span>),
whereas the return value of published&nbsp;&nbsp;functions is the data
you want to send back to the client (though you can also write on the
request object to make things more interesting).<br><br>To sum up,
handlers are the lowest API of mod_python, and the publisher is just a
handler which has its own system for publishing, which somewhat
resembles the handler system, but with enough differences to prevent
the two from being swappable at will... We know it's quite confusing
for a newbie, but now that there are users for the two layers, we can't
unify them :).<br><br>Regards,<br>Nicolas<br><br>2005/10/6, Ken Bradshaw &lt;<a href="mailto:bradshak@us.ibm.com">bradshak@us.ibm.com</a>&gt;:<br>&gt;&nbsp;&nbsp;<br>&gt; <br>&gt; Hello All,<br>&gt;&nbsp;&nbsp;I've recently taken an interest in python and mod_python in particular. For
