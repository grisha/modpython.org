<P>After further testing I have managed to find a&nbsp;very simple solution. Instead of using popen2 you can import from os&nbsp;popen and&nbsp;do a read*()&nbsp;on the standard output since you are&nbsp;creating a file object. So here's what I've done:</P>
<P>userCookie = popen(cookieGenerator).readline()</P>
