<br>
  if req.auth_type() != &#39;wgtiauth&#39;:<br>
    return apache.DECLINED<br>

to be setting it to &#39;nobody&#39; and it should really be the actual user<br>
name. That way you can then use other Apache directives such as<br>
&#39;Require user&#39;.<br>
<br>
3. If the authentication handler was successful, it should be setting<br>
req.ap_auth_type to be the authentication type.<br>
<br>
  req.ap_auth_type = req.auth_type()<br>
