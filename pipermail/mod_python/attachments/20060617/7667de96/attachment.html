There have been questions from Karl Kobata and John Cartwright and I may have a partial answer.<br>
<br>
In particular, one of the problems that I spent a little time over was
the use of the Apache &lt;Directory&gt; tag to control mod_python
configuration.<br>
<br>
The example in the mod_python tutorial is slightly ambiguous,
particularly as it is written from a Unix / Linux perspective.&nbsp; I
suspect that this is bog-standard Apache configuration rather than
anything peculiar to mod_python but I guess that people are coming
fairly new to both at the same time (as was I).<br>
<br>
The directory in the &lt;Directory&gt; tag has to be an absolute
Windows directory that should include a drive letter.&nbsp; This will
have to map to a directory under the Apache root directory.&nbsp; I was
expecting the reference to be relative to the Apache root directory.<br>
<br>
For example, suppose that you have an Apache configuration with the
Apache root set to E:\ApacheRoot and you have a sub-directory testdir
in which you have placed mod_python sources.&nbsp; Then you can refer
to URIs in this directory from the browser as
<a href="http://yourdomain/testdir/test.py">http://yourdomain/testdir/test.py</a> if you have a set of Apache
directives such as the folliowing<br>
<font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&lt;Directory
&quot;E:\ApacheRoot\testdir&quot;&gt;</span></font>

<p style="margin-left: 0.25in; text-indent: 0.25in;"><font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
AddHandler mod_python .py</span></font></p>


<p style="margin-left: 0.25in; text-indent: 0.25in;"><font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
PythonHandler mptest</span></font></p>


<p style="margin-left: 0.25in; text-indent: 0.25in;"><font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
PythonDebug On </span></font></p>


<p style="margin-left: 0.25in; text-indent: 0.25in;"><font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&nbsp;&lt;/Directory&gt;</span></font></p>
In this case, my confusion was to assume that I would have a tag as follows:<br>
<font face="Arial" size="2"><span style="font-size: 10pt; font-family: Arial;">&lt;Directory
&quot;/testdir&quot;&gt;</span></font>

<br>

<br>
BTW, if you are really struggling with set-up, I got some initial
progress by omitting the &lt;Directory&gt; tag and just having the
AddHandler etc. at the root level.&nbsp; Of course, this is not wise
for a real implementaton but, if you are trying to find out if your
mod_python installation is OK then it gave me some information.&nbsp;
of course, I had to put the .py file in the Python Lib directory in the
installation so it could be imported but that did get me some basics
working.<br>
