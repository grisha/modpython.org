Try this:

Make sure you have libtool 1.4.x, you can do this with libtool --version.

If/when you do, rebuild your apache like this:

cd httpd-2.0.43
./buildconf
./configure --prefix=/usr/local/apache2
make
make install

then ./configure and make mod_python.
