Thanks to Simon for earlier response re getting cookies.

A nasty hack that allows me to set cookies is:

     c = Cookie.SimpleCookie()
     c['fred']  = 'blahblah'
     ...
     cRaw = str(c)
     if cRaw:
         req.headers_out['Set-Cookie'] = cRaw.split('Set-Cookie: ', 1)[1]
