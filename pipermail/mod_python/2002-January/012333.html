I think you're looking for this:

req.content_type = 'text/html'
req.err_headers_out['Refresh'] = '3;http://usl/to/new/location'
#Redirect after 3 Seconds
req.send_http_header()
return apache.HTTP_MOVED_PERMANENTLY
