Matthew,

I had the DEAPI error yesterday. Running Debian (level testing).

This is what I did.

"""
./configure

edit src/Makefile and change the OPT= line to read OPT=-DEAPI

then run "make dso"

then run "make install_dso"
"""

Hope this works for you.
