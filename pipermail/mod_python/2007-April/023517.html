Raising an exception will get you there, with some help. If you wrap 
your PSP page in a publisher handler, you can create a "Abort" exception 
class that you catch silently.

class Abort(Exception): pass

try:
   PSP...
except Abort:
   pass
except:
   ... normal handling...
