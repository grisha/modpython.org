Problem described here:

  https://issues.apache.org/jira/browse/MODPYTHON-220

Where you have:

  import xyz

try replacing it with:

  from mod_python import apache
  xyz = apache.import_module("~/xyz.py")

Tell us if this work around fixes problem.

Graham
