Below is my simple input filter inputFilter.py:
-------------------------------------------------------------
def inputfilter(filter):
    f=open('filter.log','ab')#for debug
    data='start'
    while data!=None:
        try:
            data = filter.read()
            if data:
                filter.write(data)
                f.write(str(data))
        except Exception,e:
                f.write(str(e))
                data=None
     if data==None:
        filter.close()
     f.close()
----------------------------------------
httpd.conf:
----------------------------------------
<Location /test/>
     PythonInputFilter inputFilter MyInputFilter
     SetInputFilter MyInputFilter
</Location>
