Jim :

I find this directive in the manual --> PythonInterpreter

I resolved with this :

<Directory "D:/webroot">
          AddHandler mod_python .py
          PythonHandler mptest
          PythonInterpreter mptest
          PythonDebug On
</Directory>

Now i can run .py files in each direcotory under localhost.

Thanks for the help.
