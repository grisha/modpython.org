You have two options, one is assemble all the text you want to output 
in a single variable:

A = "one"
A += "two"
A += "three"
return A

or use the write statement:

req.write("one")
req.write("two")
req.write("three")

I think you may even be allowed to:

print >> req, "one"
print >> req, "two"
print >> req, "three"

Which is very similar to using the write statement.
