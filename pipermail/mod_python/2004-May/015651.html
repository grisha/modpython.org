You can "turn off" mod_python for specific files or locations by using:

 SetHandler default

If you prefer httpd.conf use <Location> directive:
<Location /python/css >
SetHandler default
</Location>

or <Files> directive:

# text files
<Files *.txt>
SetHandler default
</Files>

# gfx files
<Files ~ "\.(gif|jpe?g|png)$">
SetHandler default
</Files>


Hope this helps.
