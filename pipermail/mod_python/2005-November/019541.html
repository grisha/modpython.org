> --Code--
> #!/usr/bin/env python
> from mod_python import psp
> 
> def index(req):
> req.add_common_vars()
> req.content_type='text/html;charset=utf-8'
> tmpl = psp.PSP(req, filename='3-2.html')
> for key in req.subprocess_env.keys():
> tmpl.run(vars = {'key': key, 'env': req.subprocess_env[key]})

This is going to render the template multiple times, you probably just want:

from mod_python import psp

def index(req):
  req.add_common_vars()
  req.content_type='text/html;charset=utf-8'
  tmpl = psp.PSP(req, filename='3-2.html')
  #for key in req.subprocess_env.keys():
  #  tmpl.run(vars = {'key': key, 'env': req.subprocess_env[key]})
  tmpl.run()


> --end--
> ---template---
> <html>
> <head>
> <title>
> 3-2.py
> </title>
> </head>
> <body>
> <table >
> <td><%=key %></td>
> <td><%=env %></td>
> </tr>
> </table>
> <p>test</p>
> </body>
> </html>

Then in your template use:

<html>
  <head>
    <title>
      3-2.py
    </title>
  </head>
  <body>
    <table >
<%
for key in req.subprocess_env.keys():
      # indent
%>  
      <tr>
        <td><%=key%></td>
        <td><%=req.subprocess_env[key]%></td>
      </tr>
<%
# end for
%>
    </table>
  <p>test</p>
</body>
</html>

Watch out for indenting issues.
